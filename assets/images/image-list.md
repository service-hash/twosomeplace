## 아이콘 (현재 SVG 인라인으로 대체됨)

> 현재 헤더 아이콘들은 인라인 SVG로 구현되어 있어 별도 이미지 파일이 필요하지 않습니다.
> 디자인 확정 후 커스텀 아이콘으로 교체가 필요하면 아래 목록 참고:

| 파일명 | 설명 | 비고 |
|--------|------|------|
| `icon_back.svg` | 뒤로가기 화살표 | 헤더 좌측 |
| `icon_home.svg` | 홈 아이콘 | 헤더 좌측 |
| `icon_favorite.svg` | 즐겨찾기 별 아이콘 | 헤더 우측 |
| `icon_share.svg` | 공유 아이콘 | 헤더 우측 |
| `icon_cart.svg` | 장바구니 아이콘 | 헤더 우측, 빨간 뱃지 포함 |
| `icon_search.svg` | 검색 돋보기 아이콘 | 플로팅 버튼(FAB) 내부, 흰색 |

## 이미지 생성 가이드

### 스타일 참고
- **브랜드 컬러**: `#d42a2a` (투썸플레이스 레드)
- **배경**: 메뉴 썸네일은 밝은 배경(#f0ece8 계열) 또는 투명 배경
- **음료 스타일**: 투명 컵에 레이어드 된 음료 / 투썸 로고 머그컵
- **촬영 각도**: 정면 또는 약간 위에서 내려다보는 각도
- **매장 사진**: 투썸플레이스 간판이 보이는 외관 사진

### 포맷
- 모든 이미지는 **PNG** (투명 배경 지원)
- 메뉴 썸네일은 **정사각형** 비율
- 상품 상세 이미지는 **가로형** 비율
- 매장 사진은 **가로형** 비율

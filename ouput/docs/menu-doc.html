⚡ Quick Order

🎯 핵심 목적

매일 동일한 메뉴를 주문하는 사용자의 패턴을 바탕으로, 앱 진입 즉시 1-tap 재주문이 가능하도록 설계한 핵심 기능입니다.

⚠️ 이전의 문제점

  • 6단계 화면 전환 — 기존 투썸오더 앱에서는 재주문 시 지점 선택 → 메뉴 탐색 → 메뉴 상세 → 옵션 설정 → 주문 확인 → 결제까지 6단계의 화면 전환이 필요했음
  • 동일 탐색 과정 강제 — 매일 같은 메뉴를 주문하는 루틴 사용자에게도 매번 동일한 탐색 과정을 강제하여 주문 피로도를 높임
  • TPO 기반 추천 부재 — 출근길과 오후 등 시간대별로 다른 메뉴를 주문하는 패턴이 있지만, 이를 반영한 맞춤 추천이 없어 매번 처음부터 탐색해야 했음

💡 UX 설계 근거

  • 최상단 1-tap 재주문 카드 — 가장 최근 주문한 메뉴를 카드 형태로 최상단에 배치하여, 앱 진입 즉시 1-tap 재주문이 가능하도록 설계 — 기존 5단계를 2단계(Quick Order 탭 → 결제)로 단축
  • TPO 기반 자동 추천 — 사용자의 주문 패턴과 TPO를 분석하여 상황에 맞는 메뉴를 자동 추천 — 판단 신호: ① 주문 시간대/요일 ② 최근 30일 주문 빈도 ③ 픽업 매장/위치
  • 카드 내 즉시 의사결정 — 메뉴 썸네일·이름·옵션·가격을 카드 내에서 한눈에 확인할 수 있도록 구성하여, 별도 화면 진입 없이 즉시 의사결정 가능

📋 사용자 시나리오

출근길 루틴 주문앱 실행 → Quick Order 카드에 "아이스 아메리카노(L)" 자동 노출 → 탭 → 결제
시간대별 메뉴 전환오후 3시 앱 실행 → TPO 분석으로 "스트로베리 라떼" 추천 → 탭 → 결제
매장 변경 주문다른 매장에서 앱 실행 → 해당 매장 기반 최근 메뉴 노출 → 탭 → 결제

🕐 최근 주문 (Recent Orders)

🎯 핵심 목적

최근 주문한 메뉴 3건을 리스트 형태로 메뉴 홈에 노출하여, 과거 주문 기반 재주문의 진입 허들을 제거한 보조 재주문 경로입니다.

⚠️ 이전의 문제점

  • 별도 경로 3단계 진입 — 기존 앱에서 과거 주문 이력에 접근하려면 마이페이지 → 주문내역으로 별도 진입(3단계)이 필요하여, 주문 흐름과 분리된 경로로 접근성이 낮았음
  • 관리형 UI의 시각적 한계 — 주문내역 화면은 날짜·주문번호 중심의 관리형 UI로 설계되어, 메뉴 썸네일·옵션 정보가 부족해 "어떤 메뉴였는지" 시각적으로 즉시 파악하기 어려웠음
  • 2~3번째 선호 메뉴 경로 부재 — "며칠 전 주문했던 메뉴"를 다시 주문하려면 기억에 의존하여 전체 카테고리에서 처음부터 탐색해야 하며, Quick Order(최근 1건)만으로는 커버하지 못하는 2~3번째 선호 메뉴의 재주문 경로가 부재했음

💡 UX 설계 근거

  • 메뉴 홈 직접 노출 — Quick Order 카드 하단에 최근 주문 3건을 리스트로 노출하여, 별도 경로 없이 메뉴 홈에서 바로 과거 주문에 접근 가능하도록 설계 — 기존 3단계를 1단계(직접 탭)로 단축
  • 시각적 메뉴 식별 — 각 항목에 메뉴 썸네일·이름·옵션을 함께 표시하여, 기억에 의존하지 않고 시각적으로 원하는 메뉴를 빠르게 식별할 수 있도록 구성
  • Quick Order와 역할 분리 — 항목 탭 시 메뉴 상세 화면으로 이동하여 옵션 변경 후 주문 가능 — Quick Order(즉시 동일 재주문)와 역할을 분리하여, 옵션 수정이 필요하거나 다른 선호 메뉴를 재주문하는 경로 제공

📋 사용자 시나리오

과거 메뉴 재주문메뉴 홈 → 최근 주문 리스트에서 원하는 메뉴 탭 → 메뉴 상세에서 옵션 확인/변경 → 주문
옵션 변경 재주문최근 주문 리스트에서 메뉴 탭 → 사이즈·샷 등 옵션 변경 → 주문
Quick Order 보완Quick Order에 없는 2~3번째 선호 메뉴 → 최근 주문에서 선택 → 주문

✅ 기대 성과
  • 접근 경로 67% 단축 — 과거 주문 기반 재주문의 접근 경로를 3단계에서 1단계로 67% 단축하여 재주문 진입 허들 제거
  • 재주문 커버리지 확장 — Quick Order(1건·즉시 재주문)와 최근 주문(3건·옵션 변경 가능)의 조합으로 총 4건의 재주문 옵션을 커버하여, 단일 루틴부터 다양한 선호 메뉴까지 폭넓은 재주문 니즈 충족
  • 주문 전환율 향상 — 과거 주문 메뉴의 시각적 노출로 탐색 시간을 줄이고, 메뉴 홈 진입 직후 빠른 의사결정을 유도하여 주문 전환율 향상 기대

※ 산정 기준: 접근 경로(탭) 수 기준 — 기존 = 마이페이지 → 주문내역 → 재주문 탭(3단계), 개선 = 메뉴 홈 최근 주문 항목 탭(1단계)

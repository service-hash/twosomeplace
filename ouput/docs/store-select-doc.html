역할픽업 매장 검색 및 선택

💡 설계 의도

🎯 이 화면의 핵심 목적

사용자가 음료를 수령할 매장을 빠르게 찾고 선택할 수 있도록 검색 + 즐겨찾기 + 거리순 리스트를 결합한 매장 탐색 화면입니다.

UX 설계 근거

  • 검색 바 최상단 배치 — 매장명 또는 주소를 직접 입력하여 원하는 매장을 빠르게 검색할 수 있습니다
  • 서비스 옵션/지도 필터 — 조건별 매장 탐색이 가능하도록 필터 버튼을 검색 바 아래에 배치했습니다
  • 즐겨찾는 매장 카드 슬라이더 — 재방문 매장을 카드 형태로 우선 노출하여 빠른 접근을 제공합니다
  • 거리 표시 — 326m, 436m 등 실제 거리를 표시하여 접근성을 직관적으로 파악할 수 있습니다
  • 서비스 아이콘 — WiFi, 주차, 배달 등 매장별 부가 서비스 정보를 아이콘으로 제공합니다

사용자 시나리오

근처 매장 선택거리순 리스트에서 가까운 매장을 선택
즐겨찾는 매장 재방문즐겨찾는 매장 카드에서 바로 선택
특정 매장 검색검색 바에 매장명 입력 → 결과에서 선택

🧩 화면 구성요소

상단 헤더 (Header)

좌측에 뒤로가기(메뉴홈)와 홈 버튼, 중앙에 "주문 매장 선택" 타이틀, 우측에 장바구니 아이콘이 배치된 고정 헤더입니다.

  • 뒤로가기 버튼 → 메뉴 홈(menu.html)으로 복귀
  • 홈 버튼 → 메뉴 홈(menu.html)으로 이동
  • 장바구니 아이콘에 담긴 수량 빨간색 뱃지 표시

검색 바 (Search Bar)

헤더 아래에 위치한 둥근 모서리의 검색 입력 필드입니다. 좌측에 돋보기 아이콘이 배치되어 있으며, "매장명 또는 주소를 입력해 주세요" placeholder 텍스트가 표시됩니다.

  • 키워드 입력 시 매장 목록을 실시간 필터링
  • 둥근 모서리(border-radius: 22px) 적용

필터 바 (Filter Bar)

검색 바 아래에 위치한 필터 영역입니다. "서비스 옵션" 필터 버튼과 "지도" 버튼(아이콘 포함)이 나란히 배치되어 있습니다.

  • 서비스 옵션 버튼 — 매장 서비스 조건 필터링 (WiFi, 주차, DT 등)
  • 지도 버튼 — 지도 기반 매장 탐색 전환

즐겨찾는 매장 섹션 (Favorites Section)

"즐겨찾는 매장 3" 섹션 타이틀과 함께 카드 슬라이더 형태로 즐겨찾기한 매장들이 표시됩니다. 각 카드에는 매장명, 주소, 거리, 서비스 아이콘, 매장 사진이 포함됩니다.

  • 좌우 스와이프로 카드 전환 (총 3장)
  • 서비스 아이콘: WiFi, DT, 주차, 드라이브스루 등
  • 하단 페이지네이션 닷(3개)으로 현재 카드 위치 표시
  • 카드 너비: 전체 너비 - 40px (border-radius: 16px)

매장 리스트 (Store List)

전체 매장 목록이 리스트 형태로 표시됩니다. 각 매장 아이템에는 매장명, 주소, 거리, 서비스 아이콘, 즐겨찾기 별 버튼이 포함됩니다.

  • 3개 매장 표시: 투썸2.0안국, 종로알파빌딩점, 청계천광교점
  • 일부 매장에는 매장 이미지(64x64px)가 함께 표시
  • 즐겨찾기 별 버튼으로 등록/해제 토글
  • 거리 표시(326m, 436m 등)로 접근성 직관적 파악
  • 매장 아이템 탭 시 해당 매장 선택 후 메뉴 홈으로 복귀

🔗 인터랙션 & 화면 전환

진입 경로

  • 메뉴 홈/장바구니 하단의 "매장 변경" 버튼 클릭

매장 선택 흐름

메뉴 홈 → 하단 바 "변경" → 매장 선택 화면 → 검색/즐겨찾기/목록에서 선택 → 메뉴 홈 복귀

이 화면에서 이동 가능한 곳

매장 선택→ 메뉴 홈 (menu.html) 복귀 (매장 정보 업데이트)
즐겨찾기 별 토글→ 즐겨찾기 등록/해제
뒤로가기/홈→ 메뉴 홈 (menu.html)
장바구니 아이콘→ 장바구니 (cart.html)

⚙️ 데이터 & 기술 스펙

정적 HTML로 구현되어 있으며, 데모 매장 데이터가 하드코딩되어 있습니다.

데이터 소스정적 HTML (데모 매장 데이터 하드코딩)
즐겨찾는 매장카드 슬라이더 + 페이지네이션 닷(3개)
매장 리스트3개 매장 (투썸2.0안국, 종로알파빌딩점, 청계천광교점)
최대 화면 너비430px (모바일 최적화)

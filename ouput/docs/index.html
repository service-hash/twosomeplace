📄 프로젝트 개요

본 제안서는 투썸플레이스 모바일 주문 앱의 사용자 경험 개선을 위한 UI/UX 제안입니다. 기존 앱의 복잡한 주문 동선을 분석하고, 사용자 행동 패턴을 기반으로 재주문, 메뉴 탐색, 결제 경험을 최적화하는 방안을 제시합니다.

주문 흐름을 8단계 화면으로 구성하여, 각 화면별 문제점 진단과 개선안을 제안합니다. Quick Order 1-tap 재주문, 최근 주문 메뉴 홈 노출, 인라인 수량 조정과 디저트 추천 등 핵심 개선 포인트를 통해 주문 전환율 향상과 사용자 만족도 개선을 목표로 합니다.

⚠️ 핵심 과제

기존 앱의 복잡한 주문 동선을 단순화하고, 재주문/탐색/결제 경험을 최적화하여 사용자가 최소한의 단계로 원하는 메뉴를 주문할 수 있도록 전체 주문 여정을 재설계합니다.
